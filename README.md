## Installation

1. Clone the repo from [GitLab](https://gitlab.com/iskren97/brunchy)

```sh
git clone https://gitlab.com/iskren97/brunchy.git
```

2. Switch to **dev** branch

```sh
git checkout dev
```

4. Install all packages

```sh
 npm install
```

5. Run the application

```sh
 npm start
```
